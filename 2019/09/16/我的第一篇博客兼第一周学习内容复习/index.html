面向对象的基本概念

面向对象：java C# C++ Object-c

面向过程：C

面向过程：关注的是问题的解决步骤。 吃->狗屎

面向对象：关注的是问题解决的主体（play qipai） 狗->吃屎

面向对象：主要分析有哪些对象，每一个对象有哪些属性和行为

面向对象oop三大特性： 封装、继承、多态

封装含义：可以把某个功能块封装到一个函数里面

继承含义：是用来描述类和类之间的关系，目的是为了做一些数据的共享

多态含义：同一个行为对不同对象有不同的结果（js中不关注）

JS的面向对象不是纯正的面向对象，js不是一门面向对象的编程语言，是一门基于对象的编程语言。

JS中没有Class概念，js是通过构造函数来模拟Class的

继承的作用

当有了继承之后，我们可以把所有人的公共信息都放到Person类中，其他类只要继承了Person这个类，其他类就会自动拥有Person类的所有信息。

Person代表人这个类，他有两个属性 分别是 name 和 age

Student代表学生这个类他有一个属性 是gender

构造函数的原型属性(prototype)中，除了拥有构造函数constructor之外，还有一个proto属性，该属性指向父亲的原型对象(prototype)。

对象的proto属性指向了函数的原型，就构成了原型链

prototype的作用

5.成员的搜索规则

当获取对象属性和方法的时候，搜索规则如下：

​ 1、先从对象本身去查找有没有指定信息，如果有则直接用

​ 2、如果没有则从对象所属的函数的原型中去找，如果有则使用

​ 3、如果没有则沿着原型链继续查找，如果最终原型链中都没有，则打印undefined

我们现在知道了js中的所有对象最终都继承了Object,也就意味着所有的js对象都可以调用Object.prototype中的方法

所以我们现在知道了，toString() valueOf() hasOwnProperty() toLocaleString() 这些方法都来自于Object的原型
